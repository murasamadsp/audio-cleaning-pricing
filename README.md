# Audio Cleaning Pricing Calculator

Український калькулятор цін на обробку аудіо з динамічними формулами тарифів.

## Опис

Веб-додаток для розрахунку вартості обробки аудіо файлів з використанням різних математичних формул ціноутворення. Підтримує кілька типів тарифних планів та автоматичне оновлення курсу валют.

## Особливості

- **Динамічні формули тарифів**: Гіпербола, степенева функція, логарифмічна
- **Валюта**: Підтримка USD та UAH з автоматичним оновленням курсу
- **Інтерактивний графік**: Візуалізація кривої цін з використанням Chart.js
- **Адаптивний дизайн**: Працює на всіх пристроях
- **Теми**: Підтримка світлої та темної теми
- **Пресети**: Готові налаштування для різних стратегій ціноутворення

## Технології

- **Vite** - швидкий бандлер та dev сервер
- **Vanilla JavaScript** - без фреймворків
- **Chart.js** - для візуалізації графіків
- **CSS Custom Properties** - для темізації
- **Exchange Rate API** - для отримання курсу валют

## Формули тарифів

### Гіпербола
```
A / (1 + B × x)
```
Де x - кількість хвилин аудіо

### Степенева функція
```
A × x^(-B)
```

### Логарифмічна
```
A - B × ln(x)
```
